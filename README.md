# CYBER RACER
赛博朋克风格45°斜视角线框赛车游戏，纯HTML5单文件实现，无需依赖直接打开即可游玩。

![screenshot](https://picsum.photos/800/450)

## 游戏特性
- 🎮 **动态赛道**：2/3/4车道随机变化，搭配平滑随机弯道，驾驶体验不单调
- ⚡ **能量系统**：X键超频加速（288km/h）消耗能量，满能量可连续超频8秒；W键普通加速最高200km/h不耗能量
- 💎 **能量球道具**：道路随机刷新，拾取回复35能量，触发粒子特效
- 🔥 **Combo系统**：连续超车最高5倍得分加成，挑战更高分数
- 📈 **动态难度**：NPC 密度与车速随分数递增，威胁等级可视化，越往后越刺激
- 🎵 **音效支持**：内置Web Audio合成引擎/刹车/碰撞/拾取音效，支持选择本地音乐作为BGM，音量可调
- 📱 **双端支持**：
  - PC端：键盘操作
  - 移动端：可切换手机模式，显示大尺寸虚拟按键，支持触屏操作
- 🎨 **赛博朋克视觉**：霓虹线框风格、扫描线、故障特效、高速暗角、换道拖影、粒子爆炸
- 💾 **本地存档**：最高分自动保存到浏览器localStorage

## 操作说明
### PC键盘操作
| 按键 | 功能 |
|------|------|
| A / ← | 切换到左侧车道 |
| D / → | 切换到右侧车道 |
| W / ↑ | 普通加速（最高200km/h，不消耗能量） |
| X | 超频加速（最高288km/h，消耗能量） |
| S / ↓ | 减速制动 |
| ESC | 暂停/继续游戏 |
| Space | 开始游戏 |

### 手机模式操作
点击开始界面下方「切换手机模式」即可进入触屏操作模式，屏幕底部显示虚拟按键：
- 左下角 ← / →：换道
- 右下角 S：刹车 / W：普通加速 / X（黄色大按钮）：超频加速
- 游戏中右上角可随时退出手机模式
- 非手机模式支持滑动屏幕操作

### 其他功能
- 暂停界面可选择本地音频文件作为背景音乐，滑块调节音量
- 暂停界面可返回主菜单
- 能量条颜色随剩余能量渐变：红（<20%）→黄→青（满能量）
- 车道减少时会自动并入内侧车道，无需手动操作

## 速度说明
| 状态 | 速度 |
|------|------|
| 巡航（无操作） | 144 km/h |
| 普通加速（W/↑） | 最高200 km/h |
| 超频加速（X） | 最高288 km/h |
| 制动（S/↓） | 最低32 km/h |

## 玩法提示
1. 普通行驶不消耗能量，可长期保持200km/h巡航
2. 能量耗尽无法超频，注意拾取路上的黄色能量球补充
3. 连续3次以上超车触发Combo，得分倍数最高x5，尽量保持连续超车
4. 车道变宽/变窄、弯道都是自动适配，专心躲避车辆、收集能量球即可
5. 左车道NPC车速最快，右车道最慢，超车时注意选择车道

## 运行方式
直接用浏览器打开 `race.html` 即可游玩，推荐使用Chrome/Edge等现代浏览器获得最佳效果。
